# mikeschenkel.nl

## Getting Started

```bash
# Clone this repository
git clone git@github.com:mikeschenkel/mikeschenkel.nl.git

# Go to the working directory
cd mikeschenkel.nl

# Install dependencies
npm install

# Start development
npm start
```
